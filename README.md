# Universal Credit AI Agent

This project builds a mini AI agent to analyze the Universal Credit Act 2025 PDF.

## Setup

1. Install dependencies: `pip install -r requirements.txt`
2. Ensure Ollama is running with llama3.2 model.
3. Place the PDF at the specified path.

## Usage

Run `python main.py` to generate the final JSON report.

## Tasks

- Task 1: Extract text from PDF.
- Task 2: Summarize in bullet points.
- Task 3: Extract key sections.
- Task 4: Apply rule checks.
